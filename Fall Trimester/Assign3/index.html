What is a website?

A website is a mass conglomerate of different nodes, one of which all others stem from (the index), that communicate with each other via hyperlinks. It is essentially a graph, with each page (or node) connected with "edges." These websites can be found with a url specific to the site.

What does CSS stand for?

Cascading Style Sheets.

What are the three types of CSS?

The three main types in order of precedence are inline style, imbedded style, and external style sheets.
